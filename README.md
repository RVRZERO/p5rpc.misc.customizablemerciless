To adjust the configurations with your mod, create CustomizableMerciless.toml in your mod directory and use the layout below:\n
[Multipliers]\n
DamageTaken = 1.6\n
DamageGiven = 0.8\n
ExpWon = 1.2\n
MoneyWon = 1.2\n
WeaknessTaken = 3\n
WeaknessGiven = 3\n
CritTechTaken = 3\n
CritTechGiven = 3\n

[Others]\n
GallowsExp = true\n
ReturnSafeRoom = false\n
